## **How to set up the project locally.**

- Step-1 : Install yarn [( How to install Yarn )](https://classic.yarnpkg.com/lang/en/docs/install/#windows-stable)

  `npm install --global yarn`

- Step-2 : Install Dependencies

  `yarn`

- Step-3 : Run scripts to start the project

    `yarn start`
